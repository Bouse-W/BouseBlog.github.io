C 语言：移位运算和格式化输出 & 移位运算和赋值运算
看一个例子

unsigned char test_2 = 0x12;
unsigned int test_3 = 0X0000;

printf ("%hX", test_1 << 8);            // 输出结果是 0X500
test_3 = (test_1 << 8) + test_2;
printf ("%dx", test_3);                 // 输出结果是 0X512
test_1 <<= 8;
printf ("%c", test_1);                  // 输出结果是 ''，即 0 对应的空字符</code></pre>

由上我们得知
格式化输出时，若占位符和输出值类型不匹配，则优先进行阶段或扩展，再进行输出值的运算和输出赋值时，若左右两端类型不匹配，则优先进行截断或扩展，待位数统一后再对右端进行运算和赋值。
