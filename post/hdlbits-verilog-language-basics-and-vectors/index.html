Getting started

Fun fact: For Quartus synthesis, not assigning a value to a signal usually results in 0. This problem is actually easier than the previous one.

Intel 家的仿真器 Quartus 在综合过程中，给从未赋值的信号默认赋 0。这似乎是发现了一个获取 0 电平的捷径，但强烈不建议这么做。

Verilog Language

Basics

Input and output declarations actually declare a wire unless otherwise specified. Writing input wire a is the same as input a.

若无特殊说明，input 和 output 关键字实际上就声明了一条 wire，在写法上， input a 等价于 input wire a。

assign 语句

In a Verilog "continuous assignment" (assign left_side = right_side;), the value of the signal on the right side is driven onto the wire on the left side.

Verilog 中以 assign 定义的连续赋值语句将右边信号的值赋给左边信号。

The assignment is "continuous" because the assignment continues all the time even if the right side's value changes.

A continuous assignment is not a one-time event.

assign 定义的连续赋值语句中，赋值是“连续”发生的，就算右值改变，赋值过程也不会终止，也即左值和右值始终保持一致，就像拿导线（wire）把它们连起来了。

Note that wires are directional, so "assign in = out;" is not equivalent to "assign out = in;".

在 verilog 语法中，一个 assign 就综合出一条“有方向”的 wire，不同于物理世界两头通的“导线”。比如，“令 入 = 出” ≠ “令 出 = 入”，这又有点像软件开发时的赋值语句。

Thus, the assign statements are not creating wires, they are creating the connections between the wires that already exist.

实际上，assign 语句不定义 wire，只是定义 wire 之间的连接。

When you have multiple assign statements, the order in which they appear in the code does not matter.

并列的多条 assign 语句之间是并行关系，其实际作用效果与它们出现的顺序无关。

As you might expect, a wire cannot have more than one driver (what is its logic level if there is?), and a wire that has no drivers will have an undefined value (often treated as 0 when synthesizing hardware).

一条 wire 最多只被一个信号赋值（驱动），或说最多只能有一个输入，没有信号驱动的 wire 的值是“未定义的”（在综合时常被当作 0）

Vectors

Vectors are used to group related signals using one name to make it more convenient to manipulate. For example, wire [7:0] w; declares an 8-bit vector named w that is functionally equivalent to having 8 separate wires.

向量用来组织一些相关联的信号，将它们统一命名便于操作。比如 wire [7:0] w 声明了一个 8 位向量 w，它相当于 8 根独立的 wire。

Notice that the declaration of a vector places the dimensions before the name of the vector, which is unusual compared to C syntax. However, the part select has the dimensions after the vector name as you would expect.

声明向量时，将维度值放在向量名之前，使用向量时，将位选值放在向量名之后。比如：

// 声明
vector [7:0] w;
// 调用
assign output = w[1:0]

Vectors must be declared:

type [upper:lower] vector_name;

type specifies the datatype of the vector. This is usually wire or reg. If you are declaring a input or output port, the type can additionally include the port type (e.g., input or output) as well.

上述文段规定了 vector 型变量的声明方式，应当注意到：vector 型变量是有 datatype 的。下面是一些变量声明的例子：

// 全部引用自 HDLBits
wire [7:0] w;         // 8-bit wire
reg  [4:1] x;         // 4-bit reg
output reg [0:0] y;   // 1-bit reg that is also an output port (this is still a vector)
input wire [3:-2] z;  // 6-bit wire input (negative ranges are allowed)
output [3:0] a;       // 4-bit output wire. Type is 'wire' unless specified otherwise.
wire [0:7] b;         // 8-bit wire where b[0] is the most-significant bit.

The endianness (or, informally, "direction") of a vector is whether the the least significant bit has a lower index (little-endian, e.g., [3:0]) or a higher index (big-endian, e.g., [0:3]).

向量是有“端序”的，向量声明表达式中，数据的最低有效位有高索引还是低索引在决定了向量的“端序”，最低有效位在高索引是为“大端序”，如 wire [7:0] w;。

向量端序在声明时确定，后续使用时不允许更改或错用。

In Verilog, net-type signals can be implicitly created by an assign statement or by attaching something undeclared to a module port. Implicit nets are always one-bit wires and causes bugs if you had intended to use a vector.

Disabling creation of implicit nets can be done using the `default_nettype none directive.

直接使用未声明的变量（给它赋值或用作模块的端口参数）可能会创造一个隐式网型信号（implicit net-type signals），这很容易引发 bug。

A bitwise operation between two N-bit vectors replicates the operation for each bit of the vector and produces a N-bit output, while a logical operation treats the entire vector as a boolean value (true = non-zero, false = zero) and produces a 1-bit output.

按位操作有别于逻辑操作，如按位 OR 和逻辑 OR，前者对两个操作数的每一位两两执行或运算，后者将两个操作数各自当作一个整体进行或运算，输出 1 位运算结果。这在引入 Vector 之后很有必要作出区分。

对向量使用逻辑操作和按位操作的例题

Build a circuit that has two 3-bit inputs that computes the bitwise-OR of the two vectors, the logical-OR of the two vectors, and the inverse (NOT) of both vectors. Place the inverse of b in the upper half of out_not (i.e., bits [5:3]), and the inverse of a in the lower half.

module top_module(
input [2:0] a,
input [2:0] b,
output [2:0] out_or_bitwise,
output out_or_logical,
output [5:0] out_not
);
assign out_or_bitwise = a | b;
assign out_or_logical = a || b;
assign out_not[2:0] = ~a;	// Part-select on left side is o.
assign out_not[5:3] = ~b;	//Assigning to [5:3] does not conflict with [2:0]
endmodule

注意到：对两个同维同端序的向量进行逻辑或运算，使用了 `||` 作为运算符，直接输出一位结果。

> *The concatenation operator {a,b,c} is used to create larger vectors by concatenating smaller portions of a vector together*.

> *Concatenation needs to know the width of every component (or how would you know the length of the result?). Thus, {1, 2, 3} is illegal*.

并列运算符 `{}` 用于将几个向量拼接起来。进行拼接运算时，必须知道各向量的位宽，显式声明或标记进制即可。

> *The replication operator allows repeating a vector and concatenating them together:*
```verilog
{num{vector}};

This replicates vector by num times. num must be a constant. Both sets of braces are required.

重复运算符可以将一个向量赋值指定多次并连接起来。这在对变量进行有符号扩展时是有用的。
