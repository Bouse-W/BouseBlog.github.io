Matlab 绘制幅频特性曲线的三种方法

  1. 直接法：根据傅里叶变换定义式 F(jω)=∫−∞+∞f(t)⋅e−jωtdtF(j \omega) = \int_{- \infty}^{+ \infty} f(t) \cdot e^{-j \omega t} dtF(jω)=∫−∞+∞​f(t)⋅e−jωtdt

    • 源代码
    	clear;
	clc;
	syms t f_t F_jw

	j = sqrt(-1);

	% 时域自变量
	dt = 0.001;
	t = -pi:dt:pi;
	% 频域自变量
	w = -20:0.001:20;

	% 原始信号时域表示
	f_t = 0.5 * (1 + cos(t));

	% 傅里叶变换
	F_jw = f_t * exp(-1 * j * t' * w) * dt;

	% 图像
	figure;
	% 时域图
	subplot(2, 1, 1);
	plot(t, f_t);
	title("Time Domain");
	xlabel("t");
	ylabel("f(t)");
	% 幅频特性图
	F_jw = abs(F_jw);
	subplot(2, 1, 2);
	plot(w, F_jw);
	title("Frequency Domain");
	xlabel("w");
	ylabel("F(jw)");

    • 分析一下

      • 数字域上计算积分只能是用小量求和代劳，步长越小近似效果越好。这里是用 ∑n=0N−1f(nΔt)e−jω(nΔt)⋅Δt\sum_{n=0}^{N-1} f(n \Delta t) e^{-j \omega (n \Delta t)} \cdot \Delta t∑n=0N−1​f(nΔt)e−jω(nΔt)⋅Δt 代替 ∫−∞+∞f(t)e−jωtdt\int_{- \infty}^{+ \infty} f(t) e^{-j \omega t} dt∫−∞+∞​f(t)e−jωtdt，用向量与矩阵的点乘便利计算这个求和式：

      不难看出，频域自变量 ω\omegaω 的每个值对应 e−jtT⋅ω⃗e^{-j t^T \cdot \vec \omega}e−jtT⋅ω 中的一组列向量，与行向量 f⃗(t)\vec f(t)f​(t) 点积求和后得到一个 F(jω)F(j \omega)F(jω) 的值。

  2. 使用 for 循环：方法 1 的另一种形式而已

    • 源代码

      % 单边指数信号的频谱数值求解实现代码
clc;
syms t dt w dw n w_n f_t F_jw;             % 定义变量

j = sqrt(-1);           % 定义虚数
dt = 0.001;
t = -pi: dt :pi;          % 从 0 到 10 截取 1000 个点。注意因果信号时间起点从 0.01 开始。
f_t = 0.5 * (1 + cos(t));

N = length(t);
dw = 0.001;
w = [-20 : dw: 20]';
F_jw = zeros(length(w), 1);      % 需要定义Fw为一个向量，向量维度需要与自变量w的维度相同

for n = 1 : length(w)            % 对于每一个w，利用离散求和可以计算获得一个Fw
	w_n = w(n);
	%%-------------------------------------------------------------------------------------
	 % 编写程序。对于每一个给定的角频率w，计算积分，也就是对所有的时间对应的量进行离散求和
	 % F_jw (n) = sum (dt * f_t .* exp (-j * w_n * t));
	 F_jw (n) = dt * f_t * （exp (-j * w_n * t)');
	 %%-----------------------------------------------------------------------------------
end

subplot (3, 1, 1);
plot (t, f_t);
xlabel ('时域自变量 t');
ylabel ('时域因变量 f(t)');
grid on;
title ('时域 wave');

amplitude = abs (F_jw);   % 求幅度谱
subplot (3, 1, 2);
plot (w, amplitude)     % 展示幅度谱
xlabel ('角频率/弧度');
ylabel('幅度谱')
grid on
title ('Amplitude Spectrum')

phase=angle (F_jw);   % 求相位谱
subplot (3, 1, 3);
plot (w, phase);    % 展示相位谱
xlabel ('角频率/弧度');
ylabel('相位谱/弧度')
grid on
title ('Phase Spectrum')

  3. 先射箭后画靶：求出傅里叶变换的理论表达式 F(jω)F(j \omega)F(jω)，以 ω\omegaω 为自变量，绘制图像。

    • 源代码

      clc;
syms t dt w dw f_t F_jw;

dt = 0.001;
t = -pi : dt : pi;
f_t = 0.5 * (1 + cos(t));
subplot (211);
plot (t, f_t);
xlabel ('t');
ylabel ('f(t)');
title ('f(t) 的时域波形图');

dw = 0.001;
w = -20 : dw : 20;
F_jw = sin(pi * w) ./ (w .* (1 - w.*w));
subplot (212);
plot (w, F_jw);
xlabel ('ω');
ylabel ('F(jω)');
title ('f(t) 的幅频特性曲线');

      根据前文信号分析部分直接写出 F(jω)F(j \omega)F(jω) 的解析式，以 ω\omegaω 为自变量，像画时域图一样画图即可。这样并不好，并不是所有信号都能写出时域解析式，更多时候可能只能拿到一个时点值构成的向量，更不消说基于此求出傅里叶变化的解析式。
